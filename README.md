## Problem Statement

Build a Leap Year Checker using Java Servlets. An HTML form accepts a year, and the Servlet checks if it is a leap year using standard rules. It also lists the next 5 leap years after the entered year and handles invalid inputs.

## Technologies Used

- Java (Servlets)
- HTML, CSS (internal/inline)
- Apache Tomcat 11
- Eclipse IDE

## How to Run This Project

1. Import the project into Eclipse as a Dynamic Web Project.
2. Add Apache Tomcat as the server in Eclipse.
3. Right-click project → Run As → Run on Server.
4. Open browser and go to: http://localhost:8080/LeapYearApp/index.html

## Servlet Concept Practiced

- doGet/doPost
- Input Validation
- Basic Algorithmic Logic in Servlets
